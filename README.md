# Amazon Music Clustering

## Project Overview

Amazon Music Clustering is an Unsupervised Machine Learning project that automatically groups similar songs based on their audio characteristics.

The project uses clustering techniques like K-Means to identify hidden patterns in music data without predefined labels. By analyzing features such as danceability, energy, tempo, acousticness, and valence, the system organizes songs into meaningful clusters representing musical styles, moods, or listening behaviors.

---

# Domain

Music Analytics / Unsupervised Machine Learning

---

# Problem Statement

With millions of songs available on music streaming platforms, manually categorizing tracks becomes difficult and inefficient.

The goal of this project is to automatically group similar songs based on audio features using clustering algorithms.

This helps improve:

* Music recommendation systems
* Playlist generation
* Song discovery
* Listener behavior analysis
* Market segmentation
